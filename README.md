# Podcast-Player

A simple yet robust and accessible podcast player. No dependencies, just needs a bit more style. 

Features:
- Reads MP3 files
- Seekbar
- Time + Total Time
- Play/Pause button
- Play speed button, speeds available are x0.75, x1, x1.25, x1.5, x1.75, x2
- Forward/Backward 10S buttons
- Restart  button
- Volume control
